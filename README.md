# Another Bilibili Live Recorder
## Features
* Cross-platform
* Shell-like
* Flv metadata completion
* Super lightweight
* Auto upload (needs [BaiduPCS-Go](https://github.com/iikira/BaiduPCS-Go))
## Commands
```shell
#room
room add task_name room_id
room remove task_name
room list

#enable
enable record
enable upload

#disable
disable record
disable upload

#others
quit

#WIP
help
```
